# DebISO - Debian Remix Creator
----
# What's this?
A bash script for remixed Debian and Debian-based distro.

# Dependencies
## Debian/Ubuntu
*binutils
*debootstrap
*dosfstools
*grub-efi-amd64-bin
*grub-efi-ia32-bin
*grub-pc-bin
*mtools
*squashfs-tools
*unzip
*xorriso

## Arch
*binutils
*debootstrap
*dosfstools
*grub
*mtools
*squashfs-tools
*unzip
*xorriso
